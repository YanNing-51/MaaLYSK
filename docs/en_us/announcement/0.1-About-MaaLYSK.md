# <i class="ri-information-line"></i> About MaaLYSK

MaaLYSK is an automation assistant for **Love and Deepspace**, built on the [MaaFramework](https://github.com/MaaXYZ/MaaFramework) architecture.

## Overview

Using image recognition and simulated control, MaaLYSK automates daily tasks, event tasks, and repetitive operations in Love and Deepspace, freeing you from tedious daily check-ins.

## Key Features

- **Daily**: Stamina collection, galaxy exploration, core hunt, deepspace pursuit, memory upgrade, daily wrap-up
- **Weekly**: Claw machine, kitty cards, photo booth, journaling, secret promise, shop purchases
- **Special**: Auto-read messages, answer calls, reply to moments, Stay with Me, kitty card gacha, redemption codes, mailbox
- **Limited Events**: Event sign-ins, shop collections, mini-games, story reading (updated per version)
- **Multi-Platform**: Windows x64 / arm64, macOS x64 / arm64 (Apple Silicon), Linux (untested)

## Project Status

The project is in its **early stages**, with ongoing feature development. Some features may be unstable — feedback is welcome.
