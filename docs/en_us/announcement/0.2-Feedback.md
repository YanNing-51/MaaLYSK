# <i class="ri-bug-line"></i> Feedback

Found a bug or have a suggestion? Here's how to reach us.

## Before You Report

1. **Check the docs**: Make sure the issue isn't already documented
2. **Update to latest**: Ensure you're using the latest MaaLYSK version
3. **Reproduce the issue**: Confirm it's a consistent problem, not a one-off (network lag, emulator freeze, etc.)
4. **Try searching**: Ask [Deepseek](https://chat.deepseek.com/) or search online first

## How to Report

### GitHub Issues (Recommended)

Submit at [GitHub Issues](https://github.com/Witty36/MaaLYSK/issues).

Include: **Issue description** + **Full screenshots** + **logs** + **MaaLYSK version** + system environment.

Join the QQ feedback group: **1065676561**
