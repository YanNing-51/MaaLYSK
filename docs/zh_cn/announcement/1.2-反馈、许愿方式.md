# <i class="ri-survey-line"></i> 反馈/许愿方式

使用过程中遇到问题？想要新功能？请通过以下方式反馈/许愿。

## 反馈前请先确认

1. 查阅文档：阅读用户手册，并**完整浏览**[常见问题](/zh_cn/manual/2.2.html)，确认文档站已有内容无法解决你的问题
2. 检查版本：确保使用的是**最新版** MaaLYSK（查看文档站最顶部）
3. 尝试搜索：可先尝试导出日志，利用[MAA 日志分析器](https://mla.maafw.com/)、[Deepseek](https://chat.deepseek.com/)等分析问题，并解决（可选）
4. 复现问题：确认问题是**稳定触发**而非**偶发事件**（网络卡顿、模拟器卡死、未成功进行连接、偶尔一次的个例）

## 如何提问/反馈？

❗不要提**太过基础的问题**，例如**软件在哪里下载**

❗提问时请尽可能详细描述你所遇到的问题、你的环境、你的行为

⚠️ 未按下方【正确提问格式】提供信息（截图+日志+版本）的反馈，一概不予处理，恕不进行回复。

::: danger **错误的提问格式**

1.我的软件不动了怎么办？（没有提供任何截图、问题具体情况、出现错误前你具体在做什么）

2.软件/游戏/模拟器在哪里下载？（没有完整阅读软件文档）

3.[一张截图]，为什么报错了？（同1，虽然有截图，但具体情况一无所知）

4.[一张**手机拍屏**]，我运行xx任务为什么不动？（使用手机拍屏，全都是摩尔纹且看不清楚，如果你懒得发送截图，那即使看到了，但看不清/费眼力也没法回复你的问题哦。）

::: tip **正确的提问格式**

提问公式参考：**问题具体情况**+**完整页面截图**+**logs日志**+**MaaLYSK版本**+电脑环境+自己尝试做了什么（可选）

❗截图必须同时显示：**模拟器界面**+**MaaLYSK主窗口**（任务列表、日志和任务选项都能完整显示）

❗使用**电脑截图**而不是手机拍屏！（截图方式：QQ截图、win系统自带截图、键盘截图键截图、下载截图软件截图任选其一）

:::

- ### 导出日志(`maafw.log`)的方式

1. 打开**MaaLYSK文件夹**，先把`debug`文件夹完整删除掉
2. 打开**MaaLYSK**，复现一遍（最好进行单个任务的复现）
3. 任务结束后，点击MaaLYSK页面**导出日志**按钮，选择地方存储日志

::: details MaaLYSK页面一键导出日志

复现完毕后，也可以在**MaaLYSK页面**点击对应按钮，实现`导出日志`（debug+log文件夹）

## 问题反馈渠道

> [!CAUTION]
>
> 所有的问题反馈都需要按照上述格式，提供**截图**和**日志**！！
>
> 未按要求提出的反馈，视为**无效反馈**，恕不予处理与回复。

### GitHub Issues（推荐）

前往 [GitHub Issues· Witty36/MaaLYSK](https://github.com/Witty36/MaaLYSK/issues?q=is%3Aissue) 提交问题。

提交时请包含：

- **问题具体情况** + **完整页面截图** + **logs 日志** + **MaaLYSK 版本** + 电脑环境

### Maa恋与深空反馈许愿表

适合不太熟悉 GitHub 操作的用户，填写[Maa恋与深空反馈许愿表](https://docs.qq.com/sheet/DR3B6ZW5mU1BLeFND?electronTabTitle=Maa恋与深空反馈许愿表&isOfflineNewFileFlag=true&nlc=1&tab=000001)，并附上报错信息、截图等，并将导出的日志发送到QQ群中。

加入反馈 QQ 群：**1065676561**

### QQ 群直接反馈

群内可直接与其他猎人小姐交流反馈，看看能不能直接解决问题。

确认很多人出现类似问题，需要底层代码修正，需提供**完整页面截图**+**logs日志**。

## 新功能许愿方法

想要新功能？新功能什么时候会出？

1、不确定，看老师时间，以及是否有老师提交PR。许愿入口：填写[Maa恋与深空反馈许愿表](https://docs.qq.com/sheet/DR3B6ZW5mU1BLeFND?electronTabTitle=Maa恋与深空反馈许愿表&isOfflineNewFileFlag=true&nlc=1&tab=000001)进行许愿。

2、现在就要：快速行动起来，加入程序组，自己实现想要的功能。或：付费定制，联系程序老师为你打造专属功能。
