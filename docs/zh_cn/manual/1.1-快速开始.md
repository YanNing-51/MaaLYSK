## 模拟器设置

支持市面上大多数主流模拟器，如mumu、蓝叠、雷电等。具体支持列表参考本家[模拟器和设备支持|MAA文档站](https://docs.maa.plus/zh-cn/manual/device/windows.html)

模拟器下载完成后，需保证以下配置。具体操作参照[图文教程--模拟器设置](/zh_cn/manual/1.3.html#模拟器设置)。

- **分辨率**：设置为`900*1600`-`手机`/`竖屏`。
- **后台保活运行**：`关闭`
- 性能：根据自己的电脑配置自行决定，越高越好。~~因为恋与深空真的很卡~~
- 语言：游戏界面语言设置为`简体中文`

## 安装与权限设置

下载好压缩包后进行解压，解压后的整个文件夹即MaaLYSK的安装包。根据系统进行相关权限设置。

::: details **Windows**

下载好的安装包，解压后在文件夹内双击运行`MaaLYSK.exe`即可进入软件界面。

想要放到桌面，右键`.exe`-`发送到`-`桌面快捷方式`。

> [!CAUTION]
>
> 不要解压到C盘、桌面、D/E/F盘根目录下。（解压路径示例：`D:\MaaLSK`）

:::

:::details **MacOS**

1. 下载MaaLYSK，解压缩之后，右击文件夹，选择`服务`-`新建位于文件夹位置的终端窗口`

   > 如果没有 Mac 解压软件推荐 `Keka`，App Store 即可下载。

2. 在终端中**依次**输入指令

   ```shell
   chmod a+x MaaLYSK        #将MaaLYSK制作为可执行文件
   ./MaaLYSK                #打开MaaLYSK文件，相当于双击文件夹内MaaLYSK
   ```

3. **GateKeeper隔离解除**

   在 macOS 10.15 (Catalina) 及更高版本中，Gatekeeper 可能会阻止运行未签名的应用程序

   若弹出窗口并提示："因为Apple无法检查其是否包含恶意软件"、"无法打开，因为无法验证开发者"等错误

   在文件夹终端输入

   ```shell
   xattr -rd com.apple.quarantine ./*  
   ```

   ::: details 仍然不行

   1. 前往 `系统设置` - `隐私与安全性` - `安全性` ，选择`任何来源`

   2. 前往 `系统设置` - `隐私与安全性`，下滑至页面底部，找到`已阻止使用的应用`，点击 `仍要打开`/`仍然允许`。请设置窗口保持打开，由于涉及多个文件，需要依次进行允许。（新版本mac似乎有`全部允许`选项，可以尝试）

      > 只要隐私与安全性中出现 `仍要打开`/`始终允许`，优先点击，再选择弹窗中的选项 `打开`/`在XX中显示/打开`（不要选择`移到废纸篓`/`取消`）

   3. 再次在终端输入`./MaaLYSK`

4. 如果没有安装环境或环境安装有问题，就会出现`.Net10`的下载链接，复制链接到浏览器进行下载安装环境，重新执行`./MaaLYSK`即可

> [!WARNING]
>
> 以上步骤如果不成功，请更新到最新版本MaaLYSK后重试

:::

:::details **Linux**

同macOS，下载对应版本的压缩包，解压后运行 `MaaLYSK` 即可。

:::
